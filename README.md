# unadjusted_barker
The codes of experiments conducted for the Unadjusted Barker paper.

Experiments 1, 2, 3 are for the simulation studies of Section 6.1, 6.2, 6.3 of the paper (following the arxiv version's numbering) respectively. 

Experiment 4 is the added experiment for the reviewed version (replacing the experiment of original Section 6.1).

## Latest

The simulation code, simulated results, plotting functions, and plots are included for experiment 1 (limited) 2, 3, 4 (complete). The naming of the files should be sufficiently indicative.
