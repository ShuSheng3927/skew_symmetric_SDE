# unadjusted_barker
The codes of experiments conducted for the Unadjusted Barker paper.

Experiments 1, 2, 3 are for the simulation studies of Section 6.1, 6.2, 6.3 of the paper (following the arxiv version's numbering) respectively. 

## Latest

Experiment 4 is the new experiment where we consider the dX_t = -X_t dt + \alpha X_t dW_t model. 

Experiment 2_new and 3_new are the updated versions that include the semi-implicit Euler Maruyama (implicit drift, explicit volatility).
